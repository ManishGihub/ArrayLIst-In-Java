# 📚 ArrayList-In-Java

A collection of Java programs demonstrating various operations and use-cases of **ArrayList** from the Java Collections Framework.

This repository is designed for beginners who want to learn and practice ArrayList concepts through simple and practical examples.

---

## 🚀 Topics Covered

* Taking input and printing output using ArrayList
* Merging two ArrayLists
* Removing duplicate elements
* Finding the second largest element
* Working with 2D ArrayLists (Matrix Representation)

---

## 📂 Programs Included

### 1️⃣ Input and Output Using ArrayList

**File:** `InOut.java`

Takes user input, stores elements in an ArrayList, and prints them using different approaches.

#### Example Input

```text
3
1
2
3
```

#### Output

```text
[1, 2, 3]
1
2
3
```

---

### 2️⃣ Merge Two ArrayLists

**File:** `MergeArrayList.java`

Demonstrates how to combine two ArrayLists using the `addAll()` method.

#### Output

```text
[1, 2, 3, 4, 5, 6]
```

---

### 3️⃣ Remove Duplicate Elements

**File:** `RemoveDuplicates.java`

Uses a `HashSet` to remove duplicate values from an ArrayList.

#### Output

```text
[1, 2, 3, 4, 5]
```

---

### 4️⃣ Find Second Largest Element

**File:** `SecondLargestNo.java`

Sorts an ArrayList and finds the second largest element.

#### Output

```text
[1, 5, 7, 12, 23]
12
```

---

### 5️⃣ 2D ArrayList (Matrix Input & Output)

**File:** `TowDArrayList.java`

Demonstrates how to create, store, and display a two-dimensional ArrayList.

#### Example Input

```text
3
3
1 2 3
4 5 6
7 8 9
```

#### Output

```text
123
456
789
```

Alternative Output Formats:

```text
[[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```

```text
[1, 2, 3]
[4, 5, 6]
[7, 8, 9]
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/ArrayList-In-Java.git
```

2. Navigate to the project folder

```bash
cd ArrayList-In-Java
```

3. Compile the Java file

```bash
javac FileName.java
```

4. Run the program

```bash
java FileName
```
